# /노션 - 옵시디언 ↔ 노션 완전 동기화

옵시디언과 노션 간의 양방향 동기화를 실행합니다.

## 실행 단계

1. **옵시디언 정리**: 볼트 내 파일 자동 정리 및 재배치
2. **옵시디언 → 노션**: 모든 변경사항을 노션에 업로드
3. **노션 → 옵시디언**: 노션의 변경사항을 옵시디언에 반영 (개발 중)
4. **연결 확인**: 모든 데이터베이스 연결 상태 확인

## 동기화 대상

- 📂 진행중 폴더의 모든 프로젝트
- 📂 자료 폴더 (지식노트)
- 00-받은편지함 (일일노트)

## 사용 방법

사용자가 `/노션` 명령어를 입력하면, 마스터 동기화 스크립트를 실행해주세요.

```python
import subprocess
from pathlib import Path

vault_path = Path("C:/Users/woosol/OneDrive/documents/Obsidian Vault")
script_path = vault_path / "📂 _자동화" / "마스터_동기화.py"

result = subprocess.run(
    ['python', str(script_path)],
    capture_output=True,
    text=True,
    encoding='utf-8',
    errors='replace'
)

print(result.stdout)
```

실행 후 사용자에게 동기화 결과를 요약하여 보고해주세요.
