# AI Assistant: Intelligent Development Helper

AI 기반 개발 도우미 - 모든 질문에 답변

## 기능:

1. **코드 설명**
   - 복잡한 코드 이해하기 쉽게 설명
   - 알고리즘 동작 원리
   - 아키텍처 패턴 설명
   - 라이브러리 사용법

2. **코드 생성**
   - 요구사항 → 코드 자동 생성
   - 보일러플레이트 제거
   - 베스트 프랙티스 적용
   - TypeScript 완전 지원

3. **리팩토링 제안**
   - 코드 개선 방법
   - 성능 최적화 제안
   - 가독성 향상
   - 유지보수성 개선

4. **학습 지원**
   - 새로운 기술 학습
   - 에러 메시지 이해
   - 디버깅 방법 제안
   - 문서 찾기

## 사용 예:

```
Q: "이 코드가 뭐하는 거야?"
A: [코드 분석] → 상세 설명 + 다이어그램

Q: "useEffect를 언제 써야 해?"
A: 사용 시나리오 + 예제 코드 + 안티패턴 경고

Q: "성능을 개선하려면?"
A: 프로파일링 → 병목 발견 → 최적화 제안

Q: "이 에러 어떻게 고쳐?"
A: 에러 분석 → 원인 설명 → 해결 방법 3가지
```
