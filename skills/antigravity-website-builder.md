# antigravity-website-builder

## Overview

**"홈페이지 외주 주지 마세요!"** - Google Antigravity를 사용하면 코딩 초보자도 전문가급 웹사이트를 직접 만들 수 있습니다.

Google Antigravity는 2025년 구글 DeepMind가 공개한 차세대 AI 개발 환경으로, Visual Studio Code를 기반으로 한 지능형 에이전트 플랫폼입니다. AI가 코딩 작업을 대신 수행하여 몇 가지 명령만으로 모던한 디자인의 랜딩 페이지, 웹사이트, 웹 애플리케이션을 제작할 수 있습니다.

이 스킬은 Antigravity의 강력한 기능을 활용하여:
- 🚀 빠르게 프로토타입 제작
- 💰 외주 비용 절감 (수백만원 → 무료)
- 🎨 전문가급 디자인 자동 생성
- 🔧 실시간 피드백으로 수정

초보자도 단 몇 시간 만에 배포 가능한 웹사이트를 만들 수 있습니다.

## When to Use This Skill

### ✅ 이 스킬을 사용하세요:

- **스타트업 MVP**: 빠르게 제품 랜딩 페이지 필요
- **개인 프로젝트**: 포트폴리오, 블로그, 이력서 사이트
- **외주 대체**: 수백만원 외주 대신 직접 제작
- **프로토타입**: 아이디어 검증용 웹사이트
- **학습 목적**: 웹 개발 배우면서 실전 경험
- **소규모 비즈니스**: 회사 홈페이지, 제품 소개 사이트
- **이벤트 페이지**: 빠르게 만들어야 하는 일회성 사이트

### ⚠️ 이 스킬 대신 전통 개발 고려:

- **대규모 엔터프라이즈**: 복잡한 백엔드, 보안 요구사항 높음
- **특수 기능**: AI가 이해하기 어려운 매우 특수한 요구사항
- **레거시 통합**: 기존 복잡한 시스템과 통합 필요

### 🔀 두 가지 병행:

- Antigravity로 프론트엔드 → 백엔드는 직접 개발
- 프로토타입은 Antigravity → 실제 배포는 최적화
- 디자인은 AI → 로직은 커스터마이징

## Prerequisites

- Google Antigravity 설치 (무료)
  - macOS, Windows, Linux 지원
  - 다운로드: https://antigravity.google.com
- Chrome 브라우저 (Antigravity 확장 프로그램용)
- 만들고 싶은 웹사이트의 명확한 아이디어
- 기본적인 웹사이트 개념 이해 (선택사항)

## Instructions

### Step 1: Antigravity 설치 및 초기 설정

Antigravity를 다운로드하고 첫 프로젝트를 생성합니다.

```bash
# 1. Antigravity 다운로드
https://antigravity.google.com 접속
→ macOS/Windows/Linux 버전 다운로드

# 2. 설치 완료 후 실행
# Antigravity는 VS Code 기반이므로 UI가 친숙함

# 3. Chrome 확장 프로그램 설치 (중요!)
Chrome 웹 스토어에서 "Antigravity Extension" 검색 및 설치
→ AI가 브라우저를 컨텍스트로 인식하여 실시간 피드백 가능
```

**AI 모델 선택:**
Antigravity는 여러 최신 AI 모델을 지원합니다:
- **Gemini 3 Flash** (추천: 빠른 프로토타입)
- **Gemini 3 Pro High** (추천: 복잡한 웹사이트)
- **Claude Sonnet 4.5** (추천: 정교한 디자인)
- **Claude Opus 4.5** (고급 로직)
- GPT-OSS 120B Medium

**설정 → AI Model에서 선택**

### Step 2: 프로젝트 생성 및 요구사항 정의

새 웹사이트 프로젝트를 시작합니다.

**방법 1: 자연어 명령 (가장 쉬움)**

```
# Antigravity 에이전트 패널에서 입력
"Create a modern landing page for a SaaS productivity tool.
Include:
- Hero section with CTA button
- Features section with 3 key features
- Pricing table with 3 tiers
- Footer with social links

Design style: Minimalist, blue color scheme, professional"
```

**방법 2: 상세 요구사항 문서**

```markdown
## 웹사이트 요구사항

### 목적
AI 기반 생산성 도구 랜딩 페이지

### 페이지 구조
1. Hero Section
   - 제목: "AI로 업무 시간 50% 단축"
   - 부제목: "똑똑한 자동화로 반복 작업 제거"
   - CTA: "무료 시작하기" 버튼

2. Features (3개)
   - 자동화 워크플로우
   - 실시간 협업
   - 데이터 분석

3. Pricing
   - Free: $0/월
   - Pro: $29/월
   - Enterprise: 문의

4. Footer
   - 회사 정보
   - SNS 링크

### 디자인
- 색상: 청록색 (#06B6D4) 메인, 화이트 배경
- 폰트: 산세리프, 모던
- 스타일: 미니멀, 깔끔
- 반응형: 모바일 최적화 필수
```

### Step 3: AI 에이전트로 웹사이트 생성

Antigravity AI 에이전트가 자동으로 코드를 생성합니다.

**에이전트 작업 프로세스:**

1. **요구사항 분석** (AI가 자동 수행)
   - 사용자 입력 이해
   - 웹사이트 구조 설계
   - 필요한 컴포넌트 파악

2. **코드 생성**
   ```
   Antigravity가 자동 생성:
   - index.html (구조)
   - styles.css (스타일)
   - script.js (인터랙션)
   - assets/ (이미지, 아이콘)
   ```

3. **실시간 미리보기**
   - Chrome 확장을 통해 브라우저에 자동 렌더링
   - 변경사항 즉시 반영

**진행 상황 확인:**
```
에이전트 패널에서 실시간 로그 확인:
✓ Analyzing requirements...
✓ Generating HTML structure...
✓ Creating responsive CSS...
✓ Adding interactive elements...
✓ Preview ready!
```

### Step 4: 실시간 피드백 및 수정

생성된 웹사이트를 보면서 AI에게 수정 요청합니다.

**피드백 예시:**

```
# 색상 변경
"Change the primary color from blue to purple (#8B5CF6)"

# 섹션 추가
"Add a testimonial section with 3 customer reviews before the pricing"

# 레이아웃 조정
"Make the hero section taller, and center the content vertically"

# 애니메이션 추가
"Add smooth scroll animation when clicking navigation links"

# 반응형 수정
"The mobile menu is not working properly, fix it"

# 이미지 교체
"Replace placeholder images with actual product screenshots from /assets/screenshots/"
```

**Chrome 확장의 장점:**
- 브라우저에서 요소 클릭 → Antigravity에게 "이 부분 수정" 가능
- 실시간 변경사항 확인
- 개발자 도구 없이도 피드백

### Step 5: 고급 기능 추가

AI에게 더 복잡한 기능을 요청합니다.

**인터랙티브 요소:**
```
# 폼 검증
"Add email validation to the signup form with error messages"

# 다크 모드
"Implement a dark mode toggle in the header"

# 스크롤 애니메이션
"Add fade-in animations for sections as user scrolls down"

# 카운터 애니메이션
"Add number counting animation for statistics (0 → 10,000 users)"
```

**SEO 최적화:**
```
"Optimize the website for SEO:
- Add meta tags
- Include Open Graph tags
- Add structured data
- Optimize images
- Create sitemap.xml"
```

**성능 최적화:**
```
"Optimize website performance:
- Minify CSS and JS
- Lazy load images
- Add caching headers
- Compress assets"
```

### Step 6: 배포

완성된 웹사이트를 배포합니다.

**배포 옵션:**

   ```bash
   vercel deploy

   # 또는 AI에게 요청
   "Deploy this website to Vercel"
   ```

   ```bash

   ```

3. **GitHub Pages**
   ```bash
   "Set up GitHub Pages deployment with automatic builds"
   ```

4. **커스텀 서버**
   ```bash
   "Generate Dockerfile for deployment"
   ```

**도메인 연결:**

## Key Concepts

- **에이전트 기반 개발**: AI 에이전트가 작업을 자율적으로 수행하고 관리
- **브라우저 컨텍스트**: Chrome 확장을 통해 AI가 실제 렌더링 결과를 인식
- **자연어 프로그래밍**: 코드 대신 자연어로 웹사이트 개발
- **실시간 피드백 루프**: 즉시 수정 → 즉시 확인 → 즉시 반영
- **모던 웹 스택**: 최신 HTML5, CSS3, JavaScript ES6+ 자동 사용
- **반응형 디자인**: 모바일 우선 자동 적용

## Common Pitfalls

- **너무 모호한 요청**: "좋은 웹사이트 만들어줘" → 구체적인 요구사항 필요
- **Chrome 확장 미설치**: 확장 없으면 브라우저 컨텍스트 기능 사용 불가
- **과도한 한 번에 요청**: 단계적으로 진행하는 것이 더 효과적
- **AI 모델 미선택**: Gemini 3 Flash 추천 (속도 + 품질 밸런스)
- **피드백 없이 완성 기대**: 초기 생성물은 80% 완성도, 피드백으로 100%까지
- **이미지 직접 제공 안 함**: 플레이스홀더 대신 실제 이미지 경로 제공 필요

## Tips and Best Practices

### 효과적인 요청 작성법
1. **구체적으로**: "버튼" → "파란색 둥근 모서리 버튼"
2. **참고 사이트 제시**: "Make it similar to stripe.com homepage"
3. **우선순위 명시**: "Most important: mobile responsiveness"
4. **예시 제공**: "Like this layout: [URL]"

### 최고의 결과를 위한 워크플로우
```
1. 간단한 구조 먼저 생성 (5분)
   ↓
2. 브라우저에서 확인 및 큰 수정 (5분)
   ↓
3. 디자인 디테일 조정 (10분)
   ↓
4. 인터랙션 추가 (5분)
   ↓
5. 최적화 및 배포 (5분)
```

### AI 모델 선택 가이드
| 용도 | 추천 모델 | 이유 |
|------|----------|------|
| 빠른 프로토타입 | Gemini 3 Flash | 속도 최고 |
| 복잡한 레이아웃 | Gemini 3 Pro High | 강력한 이해력 |
| 정교한 디자인 | Claude Sonnet 4.5 | 디테일 뛰어남 |
| 고급 로직 | Claude Opus 4.5 | 복잡한 JS 로직 |

### 빠른 반복을 위한 팁
- **스크린샷 활용**: 참고 디자인 스크린샷을 Antigravity에 드래그
- **단축키 사용**: Cmd/Ctrl + K → 에이전트 패널 빠른 열기
- **버전 관리**: "Save this as version 1" → 나중에 되돌리기 가능
- **컴포넌트 재사용**: "Use the same card style from the features section"

## Antigravity vs Other Tools

| 도구 | 장점 | 단점 |
|------|------|------|
| **Antigravity** | AI 자동화, 빠름, 무료 | 복잡한 백엔드 어려움 |
